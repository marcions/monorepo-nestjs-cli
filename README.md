# Nestjs CLI monorepo boilerplate

### [Monorepo](https://github.com/mikemajesty/nestjs-monorepo) CLI

### install
  - 
    ```
    $ npm i -g @mikemajesty/monorepo-nestjs-cli
    ```
  ---
### Usage
  - 1
    ```bash
    # type cli command
    $ monorepo-nestjs-cli
    ```
  - 2
    ```bash
     # select template name
    $ monorepo-nestjs-cli
    ```
